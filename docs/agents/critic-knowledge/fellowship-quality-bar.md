# Fellowship Quality Bar for The Critic

Source: Mid-demo fellowship feedback, April 2026.

## The question The Critic must help answer

### "How do you ensure quality of UGC competing with DuoLingo (full visual team)?"

DuoLingo has a massive professional team creating polished content. We have learner-generated games. The Critic's role is the quality gate that makes this work.

**Our quality guarantee:**
1. Every game passes the 3 criteria (real-world math, math IS gameplay, must know math to win)
2. Every game passes Mr. Chesure's 2 tests (Discovery + Self-Revealing Truth)
3. Every game passes Security Guard scan
4. Every game passes The Inspector's code verification
5. Every game passes The Tester's QA check
6. Guide approves before peer sharing

**What The Critic specifically catches that DuoLingo doesn't need to worry about:**
- Quiz-wrapper games (DuoLingo IS a quiz wrapper — we can't be)
- Fake-intrinsic games (looks interactive but isn't teaching)
- Math-sprinkle games (theme disconnected from math)
- Shortcut-exploitable games (win without learning)

## Quality vs. quantity tradeoff
- Fellowship feedback implied "90% drop-off in first 30min" in gaming
- This means: a FEW excellent games > many mediocre ones
- The Critic should REJECT aggressively. Better to have 2 verified standards than 87 broken ones.
- "We threw out 92% and started over" is a STRENGTH, not a weakness

## Competitive advantage through quality
| Competitor | Quality approach | Our approach |
|---|---|---|
| DuoLingo | Professional team, polished | Agent team verification, learner ownership |
| Khanmigo | Worksheet AI | Learning Contract + 3 criteria |
| Generic ChatGPT | No quality gate | 6-agent cross-check pipeline |
| Prodigy | Fun wrapper, weak math | Math IS the gameplay (no wrapper) |
