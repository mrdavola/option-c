### Decisions to document (WHY we decided)
- **Open source (Apache 2.0):** Fellowship recommends it. Increases builder recruitment, velocity between fellows, freedom after program. THE SECRET is protected by workflow, not code secrecy.
- **No chatbot:** Children don't engage with chatbots. Scenario Gate (3-step interactive) replaces conversation-based teaching.
- **One standard at a time:** 87-game rebuild failed. Quality over quantity. Each standard gets a Learning Contract before code.
- **Agent team:** 11 specialized agents with cross-checking. Compensates for not having DuoLingo's full visual team.
