Quality Gate — every game must pass all three
🎓
Mr. Chesure
Pedagogy Guardian
Ensures every game teaches its math standard correctly, using proven methods from the Common Core Progressions Documents.

Checks: Discovery test · Self-Revealing Truth · CPA progression · Grade-appropriate rounds

⚖️
The Critic
Game Quality Judge
Evaluates every game against the 3 criteria. Detects quiz-wrappers and fake-intrinsic games. No mercy.

Checks: Real-world math · Math IS gameplay · Must know math to win

🔍
The Tester
QA Verification
Verifies games actually work. Loads, clicks, plays through, checks every round, catches bugs.

Checks: Loads correctly · Interactions work · Rounds progress · Win detection

Engagement
🎮
The Designer
Engagement & UX Expert
Ensures every game is something kids WANT to play — not just something that teaches correctly. Asks: "Would a 10-year-old choose this over Roblox?" Evaluates first impressions, engagement loops, motivation, age-appropriateness, and anti-patterns that kill fun.

Checks: First impression · Engagement loop · Age-appropriate · The Roblox test

Support Agents
🔬
The Inspector
Code Explainer & Math Verifier
Reads generated game HTML and explains exactly what it does. Verifies correct math problems, right answers, and interaction paths.

Checks: Math accuracy · Contract alignment · Shortcut analysis

🛡️
Security Guard
HTML Security Scanner
Scans all game HTML for security threats. Verifies CSP, blocks external scripts, catches phishing and crypto mining.

Checks: CSP present · No external scripts · No data exfil · Content safety

📚
The Historian
Docs · Logging · Archive
Maintains the Blueprint as source of truth. Logs events to Firebase. Tracks costs. Archives articles, screenshots, decisions.

Tracks: Blueprint updates · Decision history · Failed approaches · API costs

🧹
The Janitor
Code Cleanup
Audits codebase for dead code, unused files, inconsistent patterns. Cleans without deleting anything we might want back.

Checks: Dead files · Unused deps · Duplicates · Legacy code

📊
The Tracker
Coverage Monitor
Tracks which of 466 standards are done, which have contracts, which have games. Maintains the progress dashboard.

Tracks: 466 standards · 2 verified ✓ · 22 games planned · % complete
